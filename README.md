# UWE MSc Cyber Security - Pre-Admission Preparatory Work

Welcome to my portfolio repository for the MSc Cyber Security pre-admission work at UWE Bristol.
Prepared by: **Korawich Naikrathok**

This repository is divided into two main sections as requested:

1. **[Network Security](./1_Network_Security):** Contains traffic analysis reports, indicators of compromise (IOCs), and presentation slides for two malware infection cases (NetSupport RAT and Lumma Stealer).
2. **[IoT Systems Security](./2_IoT_Systems_Security):** Contains four C programs testing the statistical distribution and uniformity of the required Pseudo-Random Number Generator (PRNG).

Please navigate to the respective folders to view the detailed READMEs, source codes, presentation slides, and video links.
